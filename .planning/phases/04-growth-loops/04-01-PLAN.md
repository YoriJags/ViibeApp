    Create backend/app/routes/referrals.py with:

    GET /referrals/my-code (authenticated):
    - Call get_current_user; 401 if not found
    - If user.referral_code exists, return {"code": user.referral_code, "link": f"https://viibe.app/invite/{user.referral_code}"}

    - Write referral_code to db.users, return {"code": code, "link": f"https://viibe.app/invite/{code}"}

    POST /referrals/redeem (authenticated), body: {"code": str}:
    - 401 if not authenticated
    - 400 if current_user.referred_by is not None (already redeemed)
    - Find referrer = await db.users.find_one({"referral_code": code})

    - 400 if referrer["id"] == current_user.id
    - Atomically: db.users.update_one({"id": referrer["id"]}, {"$inc": {"clout_points": 50}})
    - db.users.update_one({"id": current_user.id}, {"$inc": {"clout_points": 25}, "$set": {"referred_by": referrer["id"]}})
    - Return {"message": "Referral redeemed", "clout_awarded": 25, "referrer_clout_awarded": 50}
