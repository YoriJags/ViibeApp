# Vibez Investor Demo Video

Animated investor pitch video built with [Remotion](https://remotion.dev).

## Setup

```bash
cd video
npm install
npm start          # Opens Remotion Studio in browser
```

## Scenes (120 seconds at 30fps)

| Scene | Time | Content |
|-------|------|---------|
| PreRoll | 0-5s | Logo + tagline |
| Scene 1 | 5-20s | The Hook — real-time venue intel |
| Scene 2 | 20-38s | Vibe Oracle — peak-time prediction |
| Scene 3 | 38-52s | Scout the Vibe — emoji rating UI |
| Scene 4 | 52-75s | Ask Vibe — AI concierge |
| Scene 5 | 75-90s | Vibe DNA — user fingerprint |
| Scene 6 | 90-105s | Merchant Intelligence — analytics + revenue |
| Scene 7 | 105-120s | Closing — features + investment CTA |

## Render

```bash
npm run render     # Outputs out/vibez-demo.mp4
```

## VS Code Extension

With the Remotion VS Code extension installed:
1. Open the `video/` folder in VS Code
2. Click the Remotion icon in the sidebar
3. Select `VibezDemo` composition to preview
